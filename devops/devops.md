# 20 Quick DevOps Interview Questions  for Students

## 1️ DevOps Basics

1. **What is DevOps?**
    
2. **What are the benefits of DevOps?**
    
3. **What is CI/CD?**
    
4. **What tools are used in DevOps?**  
    Example: Git, Jenkins, Docker, Kubernetes, AWS
    

---

## 2️ Linux Questions

5. **How do you check disk usage in Linux?**  
    Expected: `df -h`
    
6. **How do you check running processes?**  
    Expected: `ps`, `top`
    
7. **How do you check memory usage?**  
    Expected: `free -m`
    
8. **How do you see logs in Linux?**  
    Expected: `cat`, `less`, `tail -f`
    

---

## 3️ Networking Questions

9. **What is DNS?**
    
10. **What is the difference between TCP and UDP?**
    
11. **What is a port?**
    
12. **How do you check if a server is reachable?**  
    Expected: `ping`
    

---

## 4️ Docker Questions

13. **What is Docker?**
    
14. **What is a Docker image?**
    
15. **What is the difference between image and container?**
    
16. **How do you run a Docker container?**  
    Expected: `docker run`
    

---

## 5️ Cloud Basics (AWS)

17. **What is EC2?**
    
18. **What is S3 used for?**
    
19. **What is IAM?**
    

---

	## 6️ Scenario Question (Very Important)

20. **Your website is not opening. What will you check first?**
    

Expected answer flow:

1. Check server status
    
2. Check network connectivity
    
3. Check application service
    
4. Check logs
    
5. Check firewall / security group
    

---

# Bonus Activity (Very Useful for Interviews)

Ask one student:

**“Explain Docker in 1 minute.”**

Another student:

**“Explain CI/CD pipeline.”**

This helps them **practice communication**, which interviewers care about a lot.
